A simple desktop weather display powered by an ESP32 and a 2.42-inch display.
The device connects to Wi-Fi, retrieves live weather information, and shows it on a compact screen for easy viewing on a desk or workspace.

Features
Real-time weather updates
Wi-Fi connectivity using ESP32
Compact desktop form factor
Lightweight and low power
Customizable display layout
Hardware Used
ESP32 development board
2.42-inch display
USB power supply
Jumper wires
How It Works

The ESP32 connects to a weather API through Wi-Fi and periodically fetches weather data such as:

Temperature
Humidity
Weather conditions

The information is then displayed on the screen in a clean and simple interface.

Applications
Desk weather monitor
Smart home display
Mini IoT dashboard
Learning project for ESP32 and displays
Future Improvements
Add clock and date
Include weather icons
Support multiple cities
Add touch or button controls
Battery-powered version
