A simple desktop weather display powered by an ESP32 and a 2.42-inch display.
The device connects to Wi-Fi, retrieves live weather information, and shows it on an oled screen for easy viewing on a desk or workspace.



Fetch data like:

Temperature
Humidity
Weather conditions

The information is then displayed on the screen in a clean and simple interface.

BOM:

2.42 in SSD1309 Screen
Esp32 Dev kit
PCB (gerbers included in /pcb/gerbers)
3dp casing (in /cad)
